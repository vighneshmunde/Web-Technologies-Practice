Html

html is the standard markup language for creating web pages and web applications. It stands for HyperText Markup Language. HTML describes the structure of a web page semantically and originally included cues for the appearance of the document.
CSS CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.
